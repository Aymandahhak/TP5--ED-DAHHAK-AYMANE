# FastDelivery Microservices

## Setup
1. Install MongoDB
2. Install Node.js
3. Clone repository
4. Run `npm install` in each service directory
5. Create .env files in each service directory
6. Start all services with `node index.js`

## Services
- Authentication (Port 3004)
- Product (Port 3001)
- Order (Port 3002)
- Delivery (Port 3003)
